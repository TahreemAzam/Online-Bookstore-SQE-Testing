## Project Overview
This repository contains a Software Quality Engineering (SQE) semester project focused on testing an Online Bookstore application (BookWorms Shop).

The project evaluates the quality of the website using functional, UI, and regression testing techniques.

## Tools & Technologies
- Cypress (UI & Regression Testing)
- Selenium WebDriver (Functional Testing)
- Playwright (Parallel UI Automation)
- JavaScript, Python
- GitHub Pages (Tested Application)

## Testing Scope
- Page load validation
- Navigation bar testing
- Product listing & grid
- Filters (Categories, Price Range)
- Add to Cart functionality
- UI consistency
- Mobile responsiveness
- Regression testing

## Test Results
- Total Test Cases: 13
- Passed: 10
- Failed: 3
- Pass Rate: 77%

## Automation
Automation scripts are provided using:
- Cypress
- Selenium

Scripts are available in the `/automation` directory.
